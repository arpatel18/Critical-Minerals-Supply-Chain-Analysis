# Global Critical Minerals Supply Chain Analysis

## Executive Summary
This project analyzes global supply chains for four critical minerals—lithium, cobalt, nickel, and copper—to identify key vulnerabilities, concentration risks, and strategic sourcing implications. The analysis integrates production, trade, and geopolitical data to develop a structured view of supply chain resilience in the context of accelerating demand driven by electrification and the energy transition.

## Scope
- Lithium
- Cobalt
- Nickel
- Copper
